Email Digest Gmail設定ヘルプ ▼

Gmailで使う場合の設定手順：
  1. Googleアカウントで2段階認証を有効化
    → 2段階認証の設定
  2. アプリパスワードを生成（通常のパスワードは使えません）
    → アプリパスワードの生成
  3. 以下の値を入力：
    SMTP Hostsmtp.gmail.com
    Port587
    SMTP UserあなたのGmailアドレス
    SMTP Password手順2で生成した16文字のアプリパスワード
  4. Save → Test Send で確認
